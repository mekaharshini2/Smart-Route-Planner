Smart Route Planner

A web-based application that helps users find the shortest path between locations using graph-based algorithms.

This project allows users to dynamically add roads between different locations and calculate the most efficient route between any two points. It uses a variation of Dijkstra’s Algorithm to compute the shortest distance and display the optimal path.

Features
Add custom roads with distances
Automatically builds a graph from user input
Find shortest path between any two locations
Displays route and total distance
Clean and responsive UI
Interactive map overview of all roads

How it works
Locations are treated as nodes
Roads are treated as edges with weights (distance)
The app calculates the shortest path using a priority-based approach (Dijkstra-like logic)

Tech Stack
HTML
CSS

Use Case
